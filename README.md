## 프로젝트 개요

> 간호사의 친절한 환자 응대는 환자의 재방문 의사를 높임과 동시에 병원의 이익을 높이는 중요한 요소입니다.<br>
간호사가 환자에게 친근감을 표할 수 있는 방법 중 하나는 차트 프로그램의 기록을 활용하는 데에 있습니다.<br>
**간호사 차트 관리 프로그램은, 이러한 차트 프로그램에 기록되는 정보의 유형과 그 정보의 배치 방식의 궁금증에서 시작하였습니다.**
